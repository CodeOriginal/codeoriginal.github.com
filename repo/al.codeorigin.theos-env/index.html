Version Compatability

  • iOS 8.x

Description

  • Installs tools for developing with theos on-device.

Depends

  • Perl - installed from CoolStar's repo. Must add http://coolstar.org/publicrepo to your sources.

  • make - This is the foundation that makes theos so easy.

  • rsync - Used to sync one directory to another.

  • git - Used for version control. This is how you will actually install theos.

  • iOS Toolchain - Metapackage that installs the tools required for compilation.

  • OpenSSH - allows remote say connections.
